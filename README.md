# IT Helpdesk Ticket System

A CLI-based ticket management system inspired by Jira Service Management.

## Features
- Create tickets
- View tickets
- Update ticket status
- Set priority levels

## Tech Stack
- Python
- JSON (data storage)

## Purpose
Built to simulate real-world IT helpdesk workflows including ticket lifecycle management, prioritization, and issue tracking.

## Example Usage

1. Create a ticket
2. View all tickets
3. Update ticket status

Example:

[1] WiFi not working - Open (Medium)
[2] Password reset - Resolved (High)
